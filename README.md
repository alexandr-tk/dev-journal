# Developer Journal

A chronological log of my technical work, open-source contributions, and research.

This repository serves as a personal knowledge base and portfolio.

### Directory Structure

Files are grouped by year and month, and named according to the specific feature being built:

```text
dev-journal/
├── 2026/
│   ├── 02-Feb/
│   │   ├── blender-expose-input-visibility.md
│   │   └── blender-optimize-input-visibility.md
```
